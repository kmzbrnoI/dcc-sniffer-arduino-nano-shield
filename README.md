Arduino Nano DCC Sniffer shield
===============================

This repository contains:

1. schematic & PCB layout for Arduino Nano DCC Sniffer,
2. firmware for Arduino Nano to read packets,
3. software in python to read serial stream from Arduino Nano and store
   stream content in file.

Arduino Nano firmware is based on
[rudysmodelrailway project](https://rudysmodelrailway.wordpress.com/2015/10/23/dcc-sniffer-packet-analyser-with-arduino/).
